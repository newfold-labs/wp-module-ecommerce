# Overview

**wp-module-ecommerce** provides a brand-agnostic eCommerce experience for WordPress sites on Newfold brands. It registers with the Newfold Module Loader and integrates with wp-module-installer and wp-module-onboarding-data. Maintained by Newfold Labs. Distributed via Newfold Satis.
